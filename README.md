# testing

[![Build Status](https://travis-ci.org/therickys93/testing.svg?branch=master)](https://travis-ci.org/therickys93/testing)

Simple c library for making tests.

## Install

```
make 
make install
```

## Usage

```
#include <testing.h>

int main(void){
    assert_true(1);
    return 0;
}
```
